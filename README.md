# 艺术创想儿童绘画网站
首先是自述，对自己的项目进行简洁全面的概述，包括项目来源，项目主要内容，各个部分功能，项目完成主要针对的目标。　
用户可以为自家孩子选课、报名，校方管理员可以对报名的情况进行处理，以及对课程类别及课程的情况进行查询、修改、
搜索、统计等。主要涉及到的技术包括：html、css、JavaScript、Ajax、JQuery、Jsp、Servlet、JDBC等

网站部署在阿里云，从项目需求确定，技术方案的确定到需求细化，开发测试，版本迭代以及后期将项目部署到阿里云服务器
全部由本人设计编写。除了以上编程知识技能，由于具备较高的审美水准，我还擅长前端整体风格设计；由于带领过10人小团
队进行创业，我还擅长团队沟通协调；对用户思维有深刻的理解，能更好地运用同理心进行产品设计。
